# number-theory-calculator
A small project where I’m building a collection of number theory functions (gcd, lcm, Bézout, etc.) while practicing Java and keeping some basic tools handy to verify calculations. I’m adding things little by little, and over time I’d like it to turn into a mini toolkit. Nothing too advanced or technical — just a personal project to work on organization, good practices, and consistency.

//** Versión en Español **//:
Proyecto donde voy armando una colección de funciones de teoría de números (gcd, lcm, Bézout, etc.) mientras practico Java y tengo a mano herramientas para verificar cálculos básicos. La idea es ir subiendo cosas de a poco y, con el tiempo, convertirlo en un mini toolkit. Nada muy técnico ni avanzado; solo un proyecto personal para mejorar organización, buenas prácticas y constancia.
